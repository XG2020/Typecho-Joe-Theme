### 安装教程
- 下载后解压到`网站根目录/usr/themes/`文件加中；
- 登录到typecho后台 --> 更换外观 --> 启用
- 点击设置外观修改自己想要的设置

### 手机端下边栏修改代码
- `Typecho-Joe-Theme/public/footer.php`文件中，将 `#` 修改成自己想要的链接（不打开移动下边栏设置可以忽略）
